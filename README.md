# 💍 Wedding Event Management System

A full-stack Wedding Event Management System developed using **Node.js, Express.js, MongoDB, EJS, and MVC Architecture**.

This project helps manage wedding events, guest lists, RSVP confirmations, and gallery images through an admin dashboard.

---

## 📌 Features

### 🎨 Gallery Management

* Add Gallery Images
* Edit Gallery Images
* Delete Gallery Images
* Display Images on Website

### 📅 Event Management

* Add Wedding Events
* Edit Events
* Delete Events
* Dynamic Event Display from MongoDB

### 💌 RSVP Management

* Guest RSVP Form
* Attendance Confirmation
* View RSVP Records
* Delete RSVP Records

### 👥 Guest Management

* Add Guests
* Edit Guest Information
* Delete Guests
* Manage Table Numbers & Relations

### 📊 Dashboard

* Wedding Management Dashboard
* Quick Navigation
* Admin Management Panel

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/wedding-event-management.git
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm start
```

or

```bash
npm run dev
```

### Open Browser

```text
http://localhost:8000
```
